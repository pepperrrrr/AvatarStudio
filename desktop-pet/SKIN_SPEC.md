# 桌宠皮套交稿规范（给画师 / 自绘 / AI 出图后拆层用）

给我什么稿子，我就能独自做出什么级别的皮套：

## 路线对照

| 你提供 | 我能独自产出 | 质感 | 动作 |
|---|---|---|---|
| 什么都不给 | 官方免费皮套（Haru/Hiyori 等 20+ 款可换） | ★★★★★ 画师级 | 原生绑定全套 |
| **按下面清单拆好的分层 PNG/PSD** | **程序化绑定皮套（推荐）** | ★★★★（取决于原画） | 呼吸/眨眼/视线跟随/口型/挥手/坐站/挂饰物理 |

| 分层 PSD + 预算约 ¥1000-8000 外包绑定 | 真 Live2D（我负责约稿文档+接入） | ★★★★★ | 专业物理全套 |

## 分层清单（程序化绑定路线）

**画布**：统一尺寸（建议 2000×3000px 以上），所有图层同一坐标系对齐，透明底 PNG（或直接给 PSD）。

**必需 9 层**（缺一层少一种动作）：
1. `back-hair` 后发（头后侧整片）
2. `body` 身体+服装（不含手臂更佳）
3. `head-base` 头部底：脸+耳朵（不含五官）
4. `front-hair` 前发/刘海
5. `brow` 眉毛（左右可合一层）
6. `eye-open` 眼睛-睁（左右可合一层）
7. `eye-closed` 眼睛-闭
8. `mouth-closed` 嘴-闭
9. `mouth-open` 嘴-张

**可选增强层**（每加一层多一种表现）：
- `eye-half` 眼-半睁 → 眨眼更顺滑
- `mouth-smile` 嘴-笑 → 表情切换
- `arm-left` / `arm-right` 手臂分离 → 挥手动作
- `lower-body` 下半身单独 → 坐/站两姿
- `accessory` 辫子/挂饰/吊坠 → 弹簧物理摆动
- `blush` 腮红层 → 害羞表情

**命名**：图层名用上面的英文名（PSD 里中文名也行，注明对应关系即可）。

## 我这边负责的部分（全自动）

分层图 → Web 引擎装配（分层视差伪 3D + 骨架动画）→ 接入桌宠（点击穿透/拖拽/置顶已就绪）。
交付后想换动作参数（眨眼频率、呼吸幅度、口型灵敏度）随时调。

## 稿子哪里来

- **自绘**：iPad/PS 按清单画
- **约稿**：闲鱼/米画师，立绘约 ¥300-3000（发这份清单给画师即可）
- **AI 出图**：即梦/NovelAI/SD 生成整图，再手动粗拆层（遮挡处需补画，效果打八折）
